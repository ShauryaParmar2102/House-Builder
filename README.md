# House-Builder

## Features

- Create and edit house builds
- Select building companies dynamically from backend API
- Add optional extras with pricing catalogue
- Automatic total cost calculation (backend-driven)
- Save builds as drafts (localStorage)
- Publish builds to showcase (backend database)
- Edit and delete existing builds
- Search and sort house listings
- Dynamic UI updates using JavaScript (MVC structure)

---

## Tech Stack

- Frontend: HTML, CSS, JavaScript (Vanilla JS)
- Backend: Node.js + Express
- Data: JSON-based storage (or localStorage for drafts)
- Architecture: MVC (Model–View–Controller)

## How to Run the Project

1️⃣ Download the Project
Click the green Code button on GitHub
Select Download ZIP
Extract the ZIP file to your computer
Open the project folder in Visual Studio Code
2️⃣ Install Dependencies
Open a terminal in the project folder (where package.json is located):

npm install

This will install all required packages and create the node_modules folder.

3️⃣ Start the Server

Run the following command:

npm run start

4️⃣ Open the App

Open your browser and go to:

http://localhost:3000
